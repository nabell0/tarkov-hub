# TARKOV HUB

Escape from Tarkov 정보 사이트를 Next.js **App Router** + Tailwind CSS로 만드는 연습 프로젝트입니다.

---

### 빠른 시작

```bash
npm install
npm run dev
```

`http://localhost:3000`

---
