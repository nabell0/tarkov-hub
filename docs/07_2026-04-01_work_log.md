## 1) 공통 레이아웃/네비게이션 정리

- `app/page.tsx`에 있던 상단 헤더(타르코프 스타일 네비게이션)를 `app/layout.tsx`의 `body` 상단으로 이동
- 메인 페이지 내 중복 헤더 코드 제거
- 결과적으로 모든 페이지에서 동일 헤더를 공통 사용

## 2) 무기 상세 페이지 구성 고도화

- `M4A1`, `MP7A1` 상세 페이지 UI를 카드형 레이아웃으로 통일
- 한국어 번역 규칙 반영
  - 예: 인체공학, 수직 반동, 수평 반동, 총구 초속, 연사력, 무게, 사용 탄약
  - 부품 카테고리는 한글, 인게임 부품명은 영어 원문 유지
- 추천 탄약 섹션 추가 및 정리
  - M4A1: `M855A1`, `M856A1`
  - MP7A1: `AP SX`, `FMJ SX`

## 3) 정적 상세 페이지 -> 다이나믹 라우팅 통합

- 하드코딩 상세 페이지 제거:
  - `app/weapons/m4a1/page.tsx` 삭제
  - `app/weapons/mp7a1/page.tsx` 삭제
- 신규 다이나믹 페이지 생성:
  - `app/weapons/[weaponName]/page.tsx`
- 데이터 분리(간이 DB 역할):
  - `src/data/weapons.ts` 생성
  - 상세 페이지 데이터(스탯, 모딩, 탄약, 이미지 URL)를 `weaponsData`로 통합
- 예외 처리 추가:
  - 존재하지 않는 무기 접근 시 `무기를 찾을 수 없습니다` 렌더링
